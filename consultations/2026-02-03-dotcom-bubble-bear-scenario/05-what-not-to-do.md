# 5. 절대 하지 말아야 할 것 (Critical Don'ts)

## 5.1 투자 행동 금지 목록

### 카테고리 A: 레버리지/마진 관련 (최고 위험)

| 행동 | 위험도 | 결과 |
|------|--------|------|
| **마진 거래** | ☠️ 치명적 | 강제 청산으로 전액 손실 가능 |
| **3x 레버리지 ETF 보유** | ☠️ 치명적 | TQQQ: -99% 가능, 회복 불가 |
| **선물/옵션 투기** | ☠️ 치명적 | 원금 이상 손실 가능 |
| **암호화폐 레버리지** | ☠️ 치명적 | 24시간 청산 위험 |

```
실제 사례: 닷컴 버블 붕괴 시

마진 투자자 A:
- 2000년 3월: $100,000 투자 (50% 마진)
- 2000년 4월: NASDAQ -25% → 마진콜
- 2000년 5월: 강제 청산 → 잔액 $0

현금 투자자 B:
- 2000년 3월: $100,000 투자
- 2002년 10월: NASDAQ -78% → 잔액 $22,000
- 2015년: 원금 회복 → $100,000+
```

### 카테고리 B: 고위험 자산 보유

| 자산 | 위험도 | 회피 이유 |
|------|--------|-----------|
| **수익 없는 AI 기업** | 🔴 극고위험 | 현금소진 → 파산 |
| **IPO 직후 기업** | 🔴 극고위험 | 락업 해제 매도 폭탄 |
| **SPAC** | 🔴 극고위험 | 대부분 NAV 이하로 거래 |
| **밈 주식 (GME, AMC)** | 🔴 극고위험 | 펀더멘탈 무관 투기 |
| **페니 주식** | 🔴 극고위험 | 유동성 부족, 사기 위험 |

### 카테고리 C: 잘못된 타이밍 전략

| 행동 | 위험도 | 결과 |
|------|--------|------|
| **하락 초기 물타기** | 🟠 고위험 | -30%에서 -70%까지 추가 하락 |
| **"바닥이다" 올인** | 🟠 고위험 | 바닥 확인 불가 |
| **급등 후 추격 매수** | 🟠 고위험 | 데드캣 바운스에 당함 |
| **패닉 셀링** | 🟡 중위험 | 바닥에서 매도 |

```
물타기의 함정:

초기 투자: $100,000 @ NASDAQ 5,000
1차 물타기: $50,000 @ NASDAQ 4,000 (-20%)
2차 물타기: $50,000 @ NASDAQ 3,000 (-40%)
3차 물타기: $50,000 @ NASDAQ 2,000 (-60%)

총 투자: $250,000
평균 단가: 3,333
바닥 (NASDAQ 1,114): -67% 손실 = -$166,500

→ 물타기로 손실 절대 금액 확대
```

## 5.2 심리적 함정 경계

### "이번엔 다르다" 증후군

```
1999년: "인터넷이 모든 것을 바꾼다. 이번엔 다르다."
2007년: "집값은 절대 떨어지지 않는다. 이번엔 다르다."
2021년: "돈을 무한정 찍어도 인플레가 없다. 이번엔 다르다."
2025년: "AI가 모든 것을 바꾼다. 이번엔 다르다."

→ 모든 버블의 공통 변명
→ 결국 "이번도 같았다"
```

### FOMO (Fear Of Missing Out)

| 상황 | 잘못된 행동 | 올바른 행동 |
|------|-------------|-------------|
| 친구가 AI 주식으로 2배 벌었다 | 따라서 매수 | 본인 전략 유지 |
| SNS에서 수익 인증 봤다 | 조급하게 매수 | 생존 편향 인식 |
| "지금 안 사면 영원히 못 산다" | 올인 | 분할 매수 또는 관망 |

### 확증 편향 (Confirmation Bias)

```
❌ 위험한 행동:
- 본인 의견과 맞는 뉴스만 읽기
- 낙관론자 유튜버만 구독
- 반대 의견 무시

✅ 올바른 행동:
- 반대 의견 적극 탐색
- Bear case 시나리오 항상 고려
- 본인이 틀릴 수 있음을 인정
```

## 5.3 구체적 금지 행동 리스트

### 즉시 금지

```
1. ❌ 마진/신용 거래 중단
   - 기존 마진 포지션 즉시 정리
   - 증거금 100% 유지

2. ❌ 레버리지 ETF 매도
   - TQQQ, SOXL, UPRO 등 보유 시 즉시 매도
   - 레버리지 인버스 (SQQQ)도 마찬가지

3. ❌ 암호화폐 비중 축소
   - 포트폴리오의 5% 이하로 제한
   - 레버리지 거래 절대 금지

4. ❌ IPO/SPAC 투자 금지
   - 신규 상장 종목 최소 6개월 관망
   - SPAC 전량 매도

5. ❌ 집중 투자 금지
   - 단일 종목 10% 이상 비중 금지
   - 단일 섹터 30% 이상 비중 금지
```

### 하락장 진행 중 금지

```
1. ❌ 물타기 금지 (처음 -30%까지)
   - -30% 이전에는 추가 매수 자제
   - 현금 보존 우선

2. ❌ 바닥 선언 금지
   - "바닥이다"라고 확신하지 말 것
   - 분할 매수만 (20%, 20%, 20%...)

3. ❌ 데드캣 바운스 추격 금지
   - 급락 후 첫 반등에 매수 금지
   - 최소 2-3주 관망 후 판단

4. ❌ 뉴스에 반응 금지
   - 단기 뉴스로 매매 결정 금지
   - 계획된 전략만 실행
```

## 5.4 금지 종목/ETF 상세

### 절대 매수 금지

| 카테고리 | 예시 | 이유 |
|----------|------|------|
| **3x 레버리지** | TQQQ, SOXL, UPRO | 변동성 붕괴, 시간가치 손실 |
| **3x 인버스** | SQQQ, SDOW | 동일한 이유 |
| **밈 주식** | GME, AMC, BBBY | 펀더멘탈 무관 |
| **적자 AI 스타트업** | (비상장 다수) | 현금소진 위험 |
| **중국 ADR** | BABA, PDD | 상장폐지 + 회계 리스크 |

### 매도 고려 대상

| 종목 유형 | 기준 | 행동 |
|-----------|------|------|
| P/E 50배+ | 성장으로 정당화 안 됨 | 비중 축소 |
| 적자 기업 | 현금소진 2년 이내 | 매도 |
| Beta 1.5+ | 시장보다 1.5배 변동 | 비중 축소 |
| Debt/Equity 200%+ | 고금리 환경 취약 | 매도 |

## 5.5 정보 소스 관련 금지

### 신뢰하면 안 되는 소스

| 소스 | 위험 | 대안 |
|------|------|------|
| **유튜브 재테크** | 대부분 후행, 과장 | SEC 공시, 10-K |
| **익명 커뮤니티** | 펌핑, 사기 | 공식 애널리스트 리포트 |
| **SNS 인플루언서** | 이해충돌, 홍보 | 학술 논문, Fed 자료 |
| **무료 리딩방** | 99% 사기 | 본인 분석 |

### 신뢰할 수 있는 소스

```
✅ 공식 소스:
- SEC EDGAR (10-K, 10-Q)
- Federal Reserve 자료
- BLS (노동통계)
- 기업 IR 자료

✅ 신뢰 가능 분석:
- Bloomberg, Reuters
- Goldman Sachs, Morgan Stanley 리서치
- Bridgewater Daily Observations
- Howard Marks Memos
```

## 5.6 심리 관리 금지 행동

### 감정적 매매 금지

```
❌ 금지:
- 포트폴리오 매일 확인 (주 1회로 제한)
- 실시간 주가 알림 설정
- 하락 시 패닉 매도 충동
- 상승 시 "더 살걸" 후회

✅ 대신:
- 분기별 리밸런싱 일정 고수
- 미리 정한 규칙만 실행
- 시장 소음 차단
- 장기 관점 유지
```

### 절대 하지 말아야 할 심리적 행동

| 상황 | 금지 행동 | 이유 |
|------|-----------|------|
| 포트폴리오 -30% | "이쯤에서 회복하겠지" | 희망적 사고 |
| 친구가 수익 | "나도 똑같이 해야지" | 생존 편향 |
| 전문가 낙관론 | "전문가가 그러니까 맞겠지" | 권위 편향 |
| 손실 누적 | "본전 찾을 때까지" | 매몰 비용 오류 |

## 5.7 최종 체크리스트

### 버블 붕괴 대비 "절대 하지 말 것"

```
□ 마진/신용 거래를 하고 있지 않다
□ 레버리지 ETF를 보유하고 있지 않다
□ 암호화폐 비중이 5% 이하이다
□ 단일 종목 비중이 10% 이하이다
□ 기술주 전체 비중이 30% 이하이다
□ 수익 없는 성장주를 보유하고 있지 않다
□ 최근 IPO 종목을 보유하고 있지 않다
□ 투자 의사결정에 SNS/유튜브를 참고하지 않는다
□ 포트폴리오를 매일 확인하지 않는다
□ 본인의 투자 계획서가 문서화되어 있다
```

## 5.8 역사가 가르치는 교훈

### 닷컴 버블 피해자 공통점

```
1. "기술주는 무조건 오른다" 믿음
2. 마진으로 수익률 극대화 시도
3. 수익 없는 기업에 "미래 가치" 투자
4. -30%에서 물타기
5. -50%에서 패닉 매도
6. 바닥에서 시장 이탈
7. 회복 시 "주식은 도박" 결론

→ 원금의 80-100% 손실 후 영구 퇴장
```

### 닷컴 버블 생존자 공통점

```
1. 현금 비중 유지 (30%+)
2. 마진 사용 안 함
3. 분산 투자 (기술주 비중 제한)
4. -30%까지 관망
5. -50% 근처에서 분할 매수 시작
6. 바닥 확인 전 올인 안 함
7. 10년 이상 보유

→ 원금 보전 후 회복 시 참여
```
