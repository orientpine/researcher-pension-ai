## 2.2 섹터별 방어 전략

### 기술주 폭락 시 방어 섹터

| 섹터 | 닷컴 붕괴 시 성과 | 방어 이유 | 대표 ETF |
|------|-------------------|-----------|----------|
| **필수소비재** | +5% | 경기 무관한 소비 | XLP, VDC |
| **헬스케어** | +8% | 의료비 지출 비탄력적 | XLV, VHT |
| **유틸리티** | +12% | 규제된 수익, 고배당 | XLU, VPU |
| **통신** | -10% (제한적) | 필수 인프라 | VOX |

### 섹터별 상세 분석

#### 1) 필수소비재 (Consumer Staples)

```
강점:
- 식품, 음료, 가정용품 → 경기와 무관하게 소비
- 브랜드 파워 → 가격 전가력
- 높은 배당 수익률 (2-3%)

핵심 기업:
- Procter & Gamble (PG): 65년 연속 배당 인상
- Coca-Cola (KO): 워렌 버핏 포트폴리오 핵심
- Walmart (WMT): 경기 침체 시 오히려 수혜
- Costco (COST): 멤버십 모델로 안정 수익

ETF 추천:
- XLP (SPDR): 가장 유동적
- VDC (Vanguard): 낮은 비용 (0.10%)
- KXI (iShares Global): 국제 분산
```

#### 2) 헬스케어 (Healthcare)

```
강점:
- 노령화 + 만성질환 → 구조적 수요
- 특허 보호 → 마진 유지
- 경기 침체에도 의료비 지출 유지

핵심 기업:
- Johnson & Johnson (JNJ): 다각화된 포트폴리오
- UnitedHealth (UNH): 보험 + 서비스
- Pfizer (PFE): 배당 4%+
- Abbott (ABT): 의료기기 + 진단

ETF 추천:
- XLV (SPDR): 대형 제약사 중심
- VHT (Vanguard): 광범위 노출
- IXJ (iShares Global): 글로벌 분산
```

#### 3) 유틸리티 (Utilities)

```
강점:
- 규제된 수익률 → 안정적 현금흐름
- 높은 배당 (3-4%)
- "채권 대용" 역할

핵심 기업:
- NextEra Energy (NEE): 재생에너지 리더
- Duke Energy (DUK): 전통 유틸리티
- Southern Company (SO): 안정적 배당

ETF 추천:
- XLU (SPDR): 가장 유동적
- VPU (Vanguard): 낮은 비용
- IDU (iShares): 광범위 커버리지

주의사항:
- 금리 상승 시 약세 (금리 민감)
- 현재 고금리 환경에서는 매력 감소
```

## 2.3 경기 침체 대비 방어 포지션

### 채권 전략

#### 시나리오별 채권 선택

| 시나리오 | 추천 채권 | 이유 | ETF |
|----------|-----------|------|-----|
| **디플레이션 + 침체** | 장기 국채 | 금리 급락 시 가격 급등 | TLT, ZROZ |
| **스태그플레이션** | 단기 국채 + TIPS | 금리 변동 위험 최소화 | SHY, TIP |
| **일반 침체** | 중기 국채 | 균형잡힌 접근 | IEF, BND |

#### 채권 포트폴리오 구성안

```
버블 붕괴 대비 채권 배분 (총 채권 비중 25-30%):

┌────────────────────────────────────┐
│  단기 국채 (1-3년)     │  40%      │  SHY, VGSH
│  중기 국채 (7-10년)    │  30%      │  IEF, VGIT  
│  TIPS (물가연동)       │  20%      │  TIP, SCHP
│  투자등급 회사채       │  10%      │  LQD, VCIT
└────────────────────────────────────┘

※ 하이일드(정크본드) 회피 - 경기 침체 시 디폴트 위험
```

### 금/원자재 헤지

#### 금 (Gold) - 최후의 보루

```
닷컴 붕괴 시 금 성과:
- 2000년: $280/oz
- 2002년: $340/oz (+21%)
- 2011년: $1,900/oz (10년간 6배)

금 투자 방법:
1. 실물 금: 보관 비용, 유동성 낮음
2. 금 ETF (GLD, IAU): 가장 편리
3. 금광주 ETF (GDX): 레버리지 효과 (변동성 큼)

추천 배분:
- 보수적: 포트폴리오의 5-10%
- 극단적 비관: 15-20%
```

#### 원자재 - 선별적 접근

```
경기 침체 시 원자재 성과 (혼합적):

유가: 침체 초기 하락 → 공급 조절 후 회복
금속: 구리 등 산업금속 하락
농산물: 상대적 안정

추천:
- 원자재 전반 ETF: DJP, GSG (제한적 배분)
- 에너지: 지정학 헤지로 소량 (5% 이하)
```

## 2.4 달러 시나리오별 헤지 전략

### 달러 강세 시나리오

```
발생 조건:
- 글로벌 위기 → 안전자산 선호
- Fed 금리 고수준 유지
- 신흥국 위기

헤지 전략:
- 달러 현금 보유
- 미국 국채 비중 확대
- 신흥국 자산 축소
- 달러 강세 ETF: UUP
```

### 달러 약세 시나리오

```
발생 조건:
- Fed 급격한 금리 인하
- 미국 재정적자 우려
- 탈달러화 가속

헤지 전략:
- 금 비중 확대 (15-20%)
- 국제 채권: BNDX (환헤지 없음)
- 비달러 자산 일부 보유
- 달러 약세 ETF: UDN
```

### 통합 환헤지 전략

```
불확실성 높은 현재 상황:

추천 배분:
- 달러 자산: 60-70% (미국 국채, 달러 현금)
- 금: 10-15% (통화 중립적)
- 비달러 자산: 15-20% (분산 목적)

환율 급변 대비:
- 급격한 포지션 변경 피하기
- 양방향 헤지 유지
```

## 2.5 버블 붕괴 신호 감지 방법

### 선행 지표 체크리스트

| 지표 | 경고 수준 | 현재 상태 | 모니터링 방법 |
|------|-----------|-----------|---------------|
| **VIX (공포지수)** | 25+ 경고, 40+ 위기 | 변동적 | CBOE VIX |
| **신용 스프레드** | 급격한 확대 | 확대 중 | HYG-IEF 스프레드 |
| **IPO 시장** | 급격한 냉각 | 과열 | Renaissance IPO ETF |
| **마진 부채** | 사상 최고치 | 높음 | FINRA 데이터 |
| **내부자 거래** | 대량 매도 | 모니터링 필요 | SEC Form 4 |

### 단계별 대응 전략

```
[1단계: 경고 신호]
- VIX 20-25 지속
- IPO 시장 냉각 시작
→ 행동: 현금 비중 30%로 확대

[2단계: 위험 신호]
- VIX 25-35
- 신용 스프레드 급등
- 대형주 -10% 조정
→ 행동: 현금 비중 40%로 확대, 기술주 추가 매도

[3단계: 위기 신호]
- VIX 40+
- 시장 -20% 이상 하락
- 유동성 경색 조짐
→ 행동: 현금 50%, 장기채 매수 시작

[4단계: 바닥 탐색]
- VIX 극단적 수준 후 하락
- 대규모 항복 매도 발생
- 밸류에이션 정상화 (CAPE 20 이하)
→ 행동: 단계적 주식 매수 재개
```

## 2.6 타이밍 전략 (실행 가이드)

### 현재 시점 즉시 실행

```
Phase 1 (즉시):
1. 기술주 노출도 점검 → Magnificent 7 비중 확인
2. 현금 비중 20%까지 확보
3. 포트폴리오 레버리지 제거
4. 마진 계좌 정리

Phase 2 (1-2주 내):
1. 방어주 ETF 매수 시작 (XLP, XLV)
2. 금 ETF 5% 배분 (GLD/IAU)
3. 단기채 ETF 매수 (SHY)

Phase 3 (1개월 내):
1. 현금 비중 40%까지 확대
2. 성장주 → 가치주/배당주 전환
3. 국제 분산 점검
```

### 하락장 진행 중 대응

```
하락 -10%: 관망, 추가 매수 금지
하락 -20%: 일부 장기채 매수 (TLT 5%)
하락 -30%: 우량 배당주 분할 매수 시작
하락 -40%: 적극적 매수 (분할)
하락 -50%+: 바닥 근처, 용기 있게 매수
```
