# 1. 역사적 교훈: 닷컴버블 붕괴 (2000-2002)

## 1.1 닷컴버블 붕괴 개요

### 타임라인

| 시점 | 사건 | NASDAQ 지수 |
|------|------|-------------|
| 2000년 3월 10일 | **정점** | 5,048 |
| 2000년 4월 | 첫 번째 급락 (-25%) | 3,800 |
| 2000년 말 | 지속적 하락 | 2,500 |
| 2001년 9/11 | 추가 급락 | 1,500 |
| 2002년 10월 | **바닥** | 1,114 |
| 회복 기간 | **15년** (2015년에야 복귀) | 5,048 |

### 핵심 수치

- **NASDAQ 하락률**: -78% (정점 → 바닥)
- **S&P 500 하락률**: -49%
- **회복 소요 기간**: NASDAQ 15년, S&P 500 7년
- **사라진 기업**: 닷컴 기업의 약 50%가 파산

## 1.3 실제로 수익을 낸 자산 (2000-2002)

### 자산별 수익률

| 자산 클래스 | 2000-2002 수익률 | 이유 |
|-------------|------------------|------|
| **미국 장기 국채** | **+30%** | Fed 금리 인하 (6.5%→1%) |
| **금 (Gold)** | **+21%** ($280→$340/oz) | 안전자산 선호 |
| **부동산 (REITs)** | **+20%** | 주식 탈출 자금 유입 |
| **필수소비재** | **+5%** | 경기 방어적 특성 |
| **헬스케어** | **+8%** | 비탄력적 수요 |
| **유틸리티** | **+12%** | 배당 + 규제된 수익 |

### 채권이 승자였던 이유

```
Fed 금리 변화:
2000년 5월: 6.50% (정점)
2001년 1월: 금리 인하 시작
2003년 6월: 1.00% (바닥)

→ 금리가 5.5%p 하락하면서 채권 가격 급등
→ 장기채 (TLT 같은) +30% 수익
```

## 1.4 먼저 회복한 섹터 (2002-2005)

### 회복 순서

1. **에너지/원자재** (2002-2008)
   - 중국 산업화 수퍼사이클
   - 원유 $20 → $140/배럴
   - 광산주, 에너지주 10배 상승

2. **부동산** (2002-2006)
   - 저금리 + 주식시장 탈출 자금
   - 주택 가격 급등 → 2008년 서브프라임 위기로 연결

3. **금융** (2003-2007)
   - 저금리 환경에서 이익 증가
   - 그러나 2008년 금융위기로 폭락

4. **기술주** (2003-2007)
   - 생존 기업만 천천히 회복
   - 새로운 기술주 (Google 2004 IPO) 등장

### 회복 특징

> **주의**: 먼저 회복한 섹터가 다음 버블의 진원지가 됨
> - 닷컴 후 부동산 → 2008년 금융위기
> - 금융위기 후 기술주 → 현재 AI 버블?

## 1.6 핵심 교훈 요약

### 생존의 조건

1. **실제 현금흐름** - 매출만으로는 부족, 순이익 필요
2. **강한 대차대조표** - 부채 낮고 현금 풍부
3. **필수 인프라** - 없으면 안 되는 제품/서비스
4. **합리적 밸류에이션** - P/E 20배 이하

### 투자자를 위한 교훈

```
✅ DO:
- 버블 정점에서 현금 비중 확대
- 채권으로 금리 인하 수혜 받기
- 방어주로 하락 완충
- 바닥 근처에서 우량주 매수

❌ DON'T:
- "이번엔 다르다" 믿기
- 하락 초기에 물타기
- 수익 없는 성장주 보유
- 레버리지/마진 사용
```
