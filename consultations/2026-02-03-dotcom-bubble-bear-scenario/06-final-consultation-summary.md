# PART 4: 실행 전략

## 4.1 즉시 실행 (오늘)

```
□ 1. 현재 기술주 비중 확인
□ 2. 마진/레버리지 포지션 전량 청산
□ 3. 레버리지 ETF (TQQQ, SOXL 등) 즉시 매도
□ 4. 현금 비중 20%까지 즉시 확보
```

## 4.2 1-2주 내 실행

```
□ 5. 방어주 ETF 매수 시작 (XLP, XLV)
□ 6. 금 ETF 10% 배분 (IAU)
□ 7. 단기채 ETF 매수 (SGOV, SHY)
□ 8. 성장주 → 가치주 전환 시작
```

## 4.3 1개월 내 완료

```
□ 9. 목표 현금 비중 40% 달성
□ 10. 전체 포트폴리오 목표 배분 완료
□ 11. 리밸런싱 알림 설정 (분기별)
□ 12. VIX 알림 설정 (35, 40, 45)
```

## 4.4 버블 붕괴 시 대응 전략

```
[하락 -10%]: 관망, 추가 매수 금지
[하락 -20%]: 장기채(TLT) 5% 매수
[하락 -30%]: 대기 현금으로 우량주 분할 매수 시작
[하락 -40%]: 적극적 매수 (채권/금 일부 → 주식)
[하락 -50%+]: 용기 있게 매수 (바닥 근처)
```

# PART 5: 절대 하지 말아야 할 것

## 5.1 치명적 금지 (☠️)

| 행동 | 결과 |
|------|------|
| 마진/신용 거래 | 강제 청산으로 전액 손실 |
| 3x 레버리지 ETF 보유 | -99% 가능, 회복 불가 |
| 수익 없는 AI 기업 투자 | 현금소진 → 파산 |
| 하락 초기 물타기 | 손실 절대금액 확대 |
| "바닥이다" 올인 | 바닥 확인 불가 |

## 5.2 보유 금지 자산

```
❌ TQQQ, SOXL, UPRO (3x 레버리지)
❌ SQQQ, SDOW (3x 인버스)  
❌ GME, AMC (밈 주식)
❌ 적자 지속 AI 스타트업
❌ 최근 IPO/SPAC
❌ 중국 ADR (상장폐지 위험)
❌ 하이일드 채권 (HYG, JNK)
```

## 5.3 심리적 함정 경계

```
⚠️ "이번엔 다르다" → 모든 버블의 공통 변명
⚠️ FOMO (놓칠까 두려움) → 고점 매수의 원인
⚠️ 확증 편향 → 낙관론만 듣기
⚠️ 매몰 비용 → "본전 찾을 때까지"
```
