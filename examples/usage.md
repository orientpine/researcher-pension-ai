# 사용 예제

## 포트폴리오 생성 명령

아래 명령어와 투자자 프로필을 입력하면 멀티에이전트 시스템이 자동으로 포트폴리오를 구성합니다.

```
/investments-portfolio:portfolio-orchestrator 나를 위한 새로운 포트폴리오를 구성해줘.
```

---

## 투자자 프로필 예시

### 예시 1: 공격형 (30대, 장기투자)

| 항목 | 내용 |
|------|------|
| **연령대** | 30대 초반 |
| **직업** | 연구원 |
| **은퇴 예정** | 65세 (약 30년 후) |
| **투자 성향** | **공격형** (30년+ 장기투자) |
| **위험 수용도** | 높음 (단기 -30% 손실 감내 가능) |

### 예시 2: 안정형 (50대, 은퇴 준비)

| 항목 | 내용 |
|------|------|
| **연령대** | 50대 초반 |
| **직업** | 사무직 |
| **은퇴 예정** | 65세 (약 12년 후) |
| **투자 성향** | **안정형** (10~15년 중기투자) |
| **위험 수용도** | 낮음 (단기 -10% 이내 선호) |

### 예시 3: 균형형 (40대)

| 항목 | 내용 |
|------|------|
| **연령대** | 40대 초반 |
| **직업** | 엔지니어 |
| **은퇴 예정** | 65세 (약 22년 후) |
| **투자 성향** | **균형형** (20년+ 장기투자) |
| **위험 수용도** | 보통 (단기 -20% 손실 감내 가능) |

---

## 실행 결과

위 프로필과 함께 명령을 실행하면 아래 파일들이 자동 생성됩니다:

```
portfolios/YYYY-MM-DD-{투자성향}-{session_id}/
├── 00-macro-outlook.md        # 거시경제 전망
├── 01-fund-analysis.md        # 펀드 분석 및 추천
├── 02-compliance-report.md    # DC형 규제 준수 검증
├── 03-output-verification.md  # 환각 방지 출력 검증
└── 04-portfolio-summary.md    # 최종 포트폴리오 요약
```

실제 생성된 보고서는 [portfolios/](../portfolios/) 디렉토리에서 확인할 수 있습니다.
